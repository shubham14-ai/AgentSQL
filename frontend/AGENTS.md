# Project: AgentSQL Frontend
Stack: Next.js 16 (App Router) + React 19 + Tailwind CSS 4 + Zustand + Recharts
API: FastAPI backend at NEXT_PUBLIC_API_URL (default http://localhost:8000)
Key paths: src/app/ (routes), src/components/ui/ (primitives), src/features/ (domain), src/store/ (zustand), src/hooks/ (custom hooks), src/types/ (TS types), src/services/ (axios)
Rules: Read `node_modules/next/dist/docs/` before using Next.js APIs (breaking changes from training data). Use "use client" for interactive components.
